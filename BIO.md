# Bio (external platforms)

This is the finalized bio for use on external platforms, Upwork, LinkedIn,
freelance profiles, and anywhere else a short "overview" field is needed.

**This text is not pulled into the website itself.** The site's own About
section (`src/components/sections/JourneySection.tsx`) is a longer, more
narrative version written specifically for the site's voice, the two are
intentionally different registers for different audiences:

- This bio, short, structured, built for a profile field someone skims in seconds.
- The site's About section, longer, reflective, meant to be read in context with everything else on the page.

If either one changes, update both deliberately rather than assuming they
should match word for word.

---

## Current version

I specialize in automating operations for startups and business owners
using n8n and AI agent workflows. I streamline manual processes, eliminate
repetitive tasks, and build systems that decide and act, not just move
data between tools.

As an AI Automation Engineer, I work across workflow orchestration, AI
agent design, and process automation, bringing in Gemini, OpenAI, or
Claude wherever a step needs real judgment instead of a fixed script. The
assistant on my own portfolio site is a working example, a system I
designed and built myself, not a mockup.

---

## House rules this bio follows

- No em dashes, anywhere, on this project, ever.
- No claimed track record, client list, or completed-projects count that
  doesn't exist yet. The only proof point used is the live AI assistant on
  this site, since that's the one thing a reader can actually go click and
  verify themselves.
- No vague credential language ("extensive expertise," "years of
  experience"). Named tools (n8n, Gemini, OpenAI, Claude) instead of
  adjectives.

## If you add an Education or Certifications field elsewhere

Nothing has been drafted for this yet since there's no confirmed real
credential on file to write from. Provide the actual detail (a specific
bootcamp name, certificate, or degree) and it'll be written honestly,
specific and verifiable, not padded.
